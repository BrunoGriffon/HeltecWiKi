HT-M01S (Rev. 2.0) is a low-cost indoor LoRa gateway based on the SX1303 + SX1250 structure and powered by an ESP32 main chip, eliminating the need for a separate Linux system. It supports Wi-Fi and Ethernet connections, is powered via Type-C 5V, and features a 1.8-inch TFT display. ## Product characteristicss

-  SX1303 baseband chip
-  LoRaWAN protocol 1.0.2 Class A, Class C support
-  Support Wi-Fi or Ethernet to connect to the network
-  Support Type-C_5V power supply
-  Onboard 1.8-inch TFT display
