## Features

- Processor: Quad-core Cortex-A55, 1.8GHz
- Memory & Storage: 2GB DDR4 RAM, 32GB eMMC 5.1
- LoRa Baseband: SX1303 + SX1250 chipset
- Connectivity: Wi-Fi (2.4/5GHz), Bluetooth 4.0/BLE
- Ethernet: Gigabit (10/100/1000M)
